# ethan luc

A portfolio website with a music-driven theme system. The site's color palette, background visuals, and atmosphere are dependant on the lofi song playing. 

## Tech Stack

- **Framework:** React 19, TypeScript 5.9, Vite 8
- **Styling:** Tailwind CSS 4, PostCSS (nesting, autoprefixer)
- **Routing:** React Router 7
- **Optimization:** React Compiler via Babel preset
